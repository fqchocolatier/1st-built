# French Quarter Chocolatier

A luxury chocolate e-commerce website built with React and Tailwind CSS. Features a complete product catalog, shopping cart, and responsive design inspired by New Orleans' French Quarter.

## 🍫 Features

- **5 Complete Pages**: Home, Collection, Product Detail, About Us, Our Craft
- **60+ Products** across 11 categories with full details
- **Shopping Cart** with add/remove/update functionality
- **Responsive Design** for desktop, tablet, and mobile
- **Luxury Branding** with custom colors and typography
- **Smooth Animations** and micro-interactions

## 🚀 Quick Start

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Run Development Server**
   ```bash
   npm start
   ```
   Opens at http://localhost:3000

3. **Build for Production**
   ```bash
   npm run build
   ```

## 📦 Deployment

### Netlify (Recommended)

1. Push your code to GitHub
2. Connect your repository to Netlify
3. Set build command: `npm run build`
4. Set publish directory: `build`
5. Deploy!

The site is already configured with `netlify.toml` for seamless deployment.

### Manual Deployment

1. Run `npm run build`
2. Upload the `build` folder to your hosting service
3. Ensure your server supports client-side routing

## 🎨 Design System

- **Colors**: Brand ivory, cream, sand, gold, brown, espresso, muted
- **Typography**: Playfair Display (headings), Jost (body), Cinzel (logo)
- **Components**: Custom gradients, animations, and transitions

## 📱 Responsive Breakpoints

- **Mobile**: < 768px
- **Tablet**: 768px - 1200px  
- **Desktop**: > 1200px

## 🛒 E-commerce Features

- Product variants and pricing
- Cart persistence (localStorage)
- Toast notifications
- Mobile-responsive cart drawer
- Quantity controls
